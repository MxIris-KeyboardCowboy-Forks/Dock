# Dock

## Description

Interact with the Dock using `CoreDockSendNotification` on macOS in pure Swift.

The API couldn't be simpler.

```swift
  Dock.run(.applicationWindows)
  Dock.run(.launchPad)
  Dock.run(.missionControl)
  Dock.run(.showDesktop)
```
